# Changelog

All notable changes to ChromaView will be documented in this file.

## [0.1.0] — 2026-05-24

### Added
- Initial release
- AB1 and SCF chromatogram file parsing
- Interactive chromatogram trace visualization with PyQtGraph
- Base call display aligned with fluorescence peaks
- Quality score (Phred) bar chart
- Manual base editing with undo/redo
- Automatic low-quality end trimming (modified Mott algorithm)
- Reverse complement sequence display
- FASTA and CSV export
- Publication-quality PNG/SVG image export
- Reference sequence comparison with mismatch highlighting
- Batch file processing with quality report
- Dark and light theme support
- Dockable panels: file browser, sequence viewer, quality chart
- Nucleotide position search
- Peak height analysis and signal-to-noise metrics
- Ambiguous base detection
- Unit test suite
